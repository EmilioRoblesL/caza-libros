# CazaLibros

Aplicación web orientada a mejorar la comprensión lectora en estudiantes mediante el uso de lecturas, cuestionarios y un sistema de gamificación.

## Tecnologías utilizadas

- Node.js
- Express
- PostgreSQL
- HTML, CSS y JavaScript

## Estructura del proyecto

- backend: lógica del servidor
- frontend: interfaz de usuario
- database: contiene el archivo SQL de la base de datos

## Base de datos

El archivo de la base de datos se encuentra en:

/database/cazalibros.sql

Este archivo contiene la estructura y los datos necesarios para recrear la base de datos del sistema.
